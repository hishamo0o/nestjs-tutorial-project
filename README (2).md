# NestJS Tutorial Project: Micro-Social API

## Project Overview
This repository is a hands-on project built while learning the **NestJS** framework. The goal is to master enterprise-grade backend architecture, focusing on modularity, decorators, and TypeScript integration.

## Features & Modules
The application is structured into several functional modules:
* **Auth Module:** Handling user registration and secure authentication logic.
* **User Module:** Managing user data and core entity logic.
* **Profile Module:** Managing user-specific profile details and settings.
* **Tweet Module:** Core logic for creating and managing short-form content posts.

## Tech Stack
* **Framework:** [NestJS](https://nestjs.com/)
* **Language:** TypeScript
* **Platform:** Node.js

## Purpose
This is a **Learning Lab** project. It serves as a documentation of my progress with NestJS, demonstrating my ability to implement controllers, services, and modules in a clean, scalable way.
